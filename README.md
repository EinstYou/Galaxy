a SDL project


needed librarys:

SDL: git clone https://github.com/libsdl-org/SDL.git

SDL_image: git clone https://github.com/libsdl-org/SDL_image.git

SDL_ttf: git clone https://github.com/libsdl-org/SDL_ttf.git




build:

cmake -B build

cmake --build build


before run this Galaxy.exe file you must to copy SDL3.dll, SDL3_image.dll, SDL3_ttf.dll and you must to paste in build folder in this project
Note: You can find dll files in the library (SDL3, SD3L_image, SDL3_ttf) folders in the build folder.
